# resolution: video analyzer

[![Lint](https://github.com/mcandre/resolution/actions/workflows/lint.yml/badge.svg)](https://github.com/mcandre/resolution/actions/workflows/lint.yml) [![license](https://img.shields.io/badge/license-BSD-0)](LICENSE.md)

# SUMMARY

resolution queries video files.

# EXAMPLE

```console
$ resolution video.mp4
1280x720
```

# DOWNLOAD

```sh
git clone https://github.com/mcandre/resolution.git ~/src/github.com/mcandre/resolution
```
